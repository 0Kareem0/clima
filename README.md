🌦️ Clima — Weather App

A modern weather dashboard built with React and Tailwind CSS.
It displays current weather conditions, air quality, and forecasts in a clean and minimal UI.

🚀 Features
🌍 Search for any city
🌡️ Current temperature & conditions
💨 Wind, humidity, UV index
🌫️ Air quality indicator
📅 7-day forecast
⏰ Hourly forecast
🌅 Sunrise & sunset times
🎨 Beautiful gradient UI with responsive design
🛠️ Tech Stack
React
Tailwind CSS
JavaScript (ES6+)
